# mylangapp 🌍

A React Native / Expo app for learning everyday vocabulary across multiple languages. Pick a language, choose a category, and browse words with pronunciations — all designed for casual, real-world language learning.

## Features
- 🌐 **9 languages** — English, Armenian (Հայերեն), Georgian (ქართული), Russian (Русский), Japanese (日本語), Chinese (中文), French (Français), Spanish (Español), Kinyarwanda (Ikinyarwanda)
- 📂 **8 vocabulary categories** — Greetings, Fun, Expressions, Food & Drink, Shop, Travel, Commands & Requests, Forms of Address
- 🔤 **Pronunciation guides** — every word includes a phonetic hint
- 🌍 **Live translation** — non-Russian languages fetched via LibreTranslate API at runtime
- 💾 **In-session caching** — translations cached during your session
- 📱 **Android-first** — built and tested on Android via Expo
